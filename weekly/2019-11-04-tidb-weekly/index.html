Updates of the week

Progress

Last week, we landed 100 PRs in the TiDB repository, 6 PRs in the TiSpark repository, and 47 PRs in the TiKV and PD repositories.

Critical PRs

TiDB:

  • Fix the potential goroutine leak when killing the connection
  • Add more fields in the slow_query table
  • Add the isolation read with the storage engine type
  • Add the push selection below projection transformation rule in the cascades planner
  • Support querying the CPU, memory, mutex, block, allocs, or goroutines flamegraphs by SQL statements
  • Stop updating the pessimistic transaction lock's TTL when the session is killed
  • Support scanning the TiFlash table in the cost-based optimizer
  • Add enable-timestamp and enable-error-stack for the log configuration
  • Support creating the TiFlash replica of a table in DDL operations

TiSpark:

  • Fix the parse failure on the JSON value of view tables

TiKV and PD:

  • Remove namespace
  • Change the configuration names from disable-* to enable-*
  • Improve the performance of the Coprocessor table lookup
  • Reject commit requests with commit_ts < primary_lock.min_commit_ts
  • Add the GetRegionCount interface
  • Keep the pessimistic lock's TTL if it's greater than the prewrite request's TTL
  • Add @winkyao to MAINTAINERS.md
