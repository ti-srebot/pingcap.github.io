TiDB TechDay 2019

为了让更多的社区伙伴能够近距离与我们展开交流，并快速 Get TiDB 3.0 GA 的技术细节和正确使用姿势，我们启动「TiDB TechDay 2019 全国巡讲」，打破「一年一城」的传统，巡回北京、上海、成都、深圳、武汉、杭州 6 座城市，用一整天的时间为当地朋友深入拆解 TiDB 3.0 以及今年技术层面的各个大招展示：从 TiDB 最新的 OLAP 架构，到云原生 TiDB demo、TiKV 性能大幅提升等等。各地用户伙伴也会一起交流分享 TiDB 实践经验，另外关于全球开源社区运营，我们又有了新的想法，也将与各地的社区伙伴们聊聊。相信在社区伙伴们的力量加持下，我们可以接着做更多做不到的事情。期待与大家见面～

2019.06-23 10:00 - 16:50
北京 · 朝阳区西大望路 27 号 灿空间
9:00 - 10:00 签到
10:00 - 10:20 Opening：畅想未来的数据库 TiDB 从出生的第一天起就希望走国际化的路线，因为我们相信只有极致开放才是构建顶级基础软件产品的唯一出路，过去一年里，TiDB 开始走出国门走向世界。本次分享将会回顾一下过去一年来，TiDB 在全球社区的一些进展。 黄东旭 PingCAP | 联合创始人兼 CTO
10:20 - 11:00 全面解析 TiDB 3.0： 易用性， 扩展性， 性能 TiDB 3.0 计划于 6 月底发布第一个稳定版。 相比 2.1， 3.0 无论是在功能上， 还是性能上都有很大进步。 希望借助这次大会和大家聊一聊 3.0 的主要特性及其背后的考量， 同时也会和大家交流一下我们对 TiDB 后续演进方向的思考。 申砾 PingCAP | Engineering VP
11:00 - 11:40 不止于代码：TiDB 社区发展及生态构建 随着 TiDB 项目的日益健壮，越来越多的全球小伙伴从各个维度加入到社区的建设中来。我会分享一下这半年来 TiDB 开源社区的那些有意思的故事，也会介绍一下在代码贡献之外还有哪些深度参与开源的方式，以及为了反哺社区我们最近的新动作。 崔秋 PingCAP | 联合创始人
11:40 - 12:10 微众银行 TiDB 实践 主要介绍微众银行的数据库架构与痛点、TiDB 引入的过程、应用场景以及问题分享，以及 WeBank 和 PingCAP 的战略合作项目。 胡盼盼 微众银行 | 数据平台室室经理

14:10 - 14:40 TiDB 在知乎万亿量级业务数据下的实践和挑战 随着用户和内容分发量的持续增长，知乎首页每日新增的阅读历史已达 30 亿，累计的历史纪录数量也已达万亿量级。在满足首页苛刻响应时间的前提下，稳定可靠的支撑海量且快速增长的数据集存在着巨大的挑战。本次分享将介绍知乎已读服务的演进以及整体迁移 TiDB 全面云化过程中所遇到的问题和挑战，以及我们对 TiDB 3.0 新特性的使用经验和感受。 孙晓光 知乎 | 搜索后端负责人
14:40 - 15:20 云原生：TiDB 的最佳使用姿势 TiDB 自诞生以来就被打上 Cloud Native 的标签，只有在云上才能发挥 TiDB 水平伸缩的真正威力。在这个环节我们将向大家演示TiDB 在云上的快速部署和管理。同时，我们会跟大家探讨 Kubernetes 和 TiDB Operator 的新特性和技术实现细节，和接下来要做的事情和一些想法。
