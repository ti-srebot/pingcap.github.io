  • 深圳华创智信科技有限公司自主研发了快速消费品行业的数字化供应链管理。
  • 该数字化供应链管理覆盖了快速消费品行业的原料生产、加工、包装、运输、分销、销售到售后服务等的大部分环节，帮助企业了解和把握需求信号和及时作出精确的预测，对需求进行分析和制定出可行的需求计划，并迅速地对需求信号作出反应，从而提高整个供应链的效率与效益。
  • 改造前系统架构 - 采用 MySQL 分库分表
  • 采用 MySQL 中间件进行分库分表。分库分表后，跨库一致性问题，都是使用最终一致性，代码维护繁琐；表加字段麻烦，每个库每个表都要加；新增节点需要进行停机、然后迁移数据，停机迁移对线上用户造成实时的读写影响。
  • 改造后的系统架构 - 引入 TiDB 替换 MySQL
  • 为从根本上解决以上问题，技术团队决定通过增加部署一套高性能的数据库系统，以解决当前业务的痛点。 最终从满足业务需求、降低切换成本和减少运维成本等角度考虑，选择了分布式关系型数据库－TiDB 作为业务的首选事务型数据库。
